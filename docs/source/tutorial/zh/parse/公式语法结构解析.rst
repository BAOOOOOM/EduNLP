公式语法结构解析
--------------------

可以检查公式是否合法。

Examples：
::

 >>> text = '支持公式如$\\frac{y}{x}$，$\\SIFBlank$，$\\FigureID{1}$，不支持公式如$\\frac{ \\dddot y}{x}$'
 >>> text_parser = Parser(text)
 >>> text_parser.description_list()
 >>> text_parser.fomula_illegal_flag
 >>> 1
